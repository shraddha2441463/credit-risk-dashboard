Corporate Credit Risk Scoring Dashboard

A web-based system to evaluate corporate financial health using Altman Z-Score, liquidity ratios, solvency ratios, and probability of default analytics.

Get Started

Upload Financial Statement CSV

Upload a standardized CSV file containing the company’s financial statement values.

Required CSV Fields

  • Total_Assets
  • Current_Assets
  • Total_Liabilities
  • Current_Liabilities
  • Total_E
  • Total_Sales
  • Working_Capital
  • Retained_Earnings
  • Market_Value_Equity
  • Total_Debt
  • Interest_Expense

Risk Score Dashboard Overview

This module will generate:

  • Altman Z-Score
  • Liquidity Ratios (Current Ratio, Quick Ratio)
  • Solvency Ratios (Debt-to-Equity, Interest Coverage)
  • Probability of Default (Low / Medium / High)
  • Graphs – Financial Trends & Ratio Comparisons

PDF Report Generation

The system will generate a downloadable PDF containing:

  • Company Name
  • All financial ratios
  • Z-score interpretation
  • Risk zone classification
  • Graphs (trend + comparison)

About This Project

This project demonstrates how financial statements can be used to assess the probability of corporate default. It integrates financial modeling with technical implementation using HTML, CSS, and Python.

  • Financial Ratio Interpretation
